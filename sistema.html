El Sistema de Información de Fundación ARU




Antecedentes.

Desde el 2012 Fundación ARU viene desarrollando un Sistema de Información Estadística interno que permita mejorar los procesos de Recolección y Análisis de los datos disponibles, el sistema es implementado y retro alimentado en cada uno de los procesos de recolección de información que se realiza y usa herramientas de Software libre, las herramientas son de tipo cartográficas, análisis de datos, de entrada de datos y diseminación.


El Concepto

El objetivo principal del sistema es garantizar la integridad de los datos y sobre todo el de obtener información pertienente y de calidad. El Sistema inicia con la entrada de la información mediante el proceso de transcripción (método tradicional y poco pertinente, cada vez lo usamos menos) o un proceso basado en el metodo CAPI (Entrevistas asistidas mediante un ordenador), luego la información pasa por un filtro de consistencia que se reporta a los usuarios del sistema, el siguiente paso depura la base de datos eliminando información inconsistente, se aplican los criterios definidos en el diseño (factores de expansión, calculo de indicadores, etc), se genera una base de datos disponible para el análisis y finalmente se genera un reporte que se encuentra en linea.

Es importante tener presente que esta lógica de trabajo se realiza de manera constante y en paralelo con la recolección de información.


Herramientas

Las herramientas empleadas para el funcionamiento del sistema son:

  1. Open Data Kit (ODK): ODK es una suite de código abierto de herramientas que ayuda a las organizaciones e investigadores a implementar el método CAPI, permite administrar soluciones de recopilación de datos móviles. Los objetivos que tiene es el de hacer herramientas de código abierto y estándares fáciles de tratar, fácil de usar, fácil de modificar y fácil de escalar. Es desarrollado por el Consorcio OpenROSA y participantes activos en el proyecto JavaRosa
  2. CSPro : El Sistema de Procesamiento de Censos y Encuestas (Census and Survey Processing System, CSPro) es un paquete de software de dominio público utilizado por cientos de organizaciones y decenas de miles de personas para la entrada, edición, tabulación y difusión de los datos de censos y encuestas. CSPro está diseñado para ser tan fácil de usar como sea posible, pero lo suficientemente potente como para manejar las aplicaciones más complejas. Es desarrollado por la centro de Censos de Estados Unidos.
  3. QGIS: QGIS es un Sistema de Información Geográfica (SIG) de Código Abierto licenciado bajo GNU - General Public License . QGIS es un proyecto oficial de Open Source Geospatial Foundation (OSGeo). Corre sobre Linux, Unix, Mac OSX, Windows y Android y soporta numerosos formatos y funcionalidades de datos vector, datos ráster y bases de datos.
  4. R CRAN: R es un lenguaje y entorno para computación y gráficos estadísticos. Es un proyecto GNU, que es similar al lenguaje S y el medio ambiente que se ha desarrollado en los Laboratorios Bell (antes AT & T, ahora Lucent Technologies) por John Chambers y colegas.
  5. REDATAM: Redatam es el acrónimo de Recuperación de Datos para Áreas pequeñas por Microcomputador. El programa Redatam+SP (abreviado R+SP), utiliza una base de datos jerárquica comprimida, crea información agregada de registros de personas, viviendas, manzanas de ciudades o cualquier división administrativa. Es posible definir, a partir de la base de datos, cualquier área geográfica de interés o combinaciones de esas áreas, crear nuevas variables, obtener varios tipos de tabulados muy rápidamente y exportar salidas a otros softwares. Los datos de diferentes niveles geográficos pueden ser combinados jerárquicamente para crear variables agregadas y los resultados pueden desplegarse en mapas, tablas, gráficas desde Redatam o transferirse a un Sistema de Información diferente para su uso.
